# 大三课程 / Junior Year Courses

## 📚 课程列表 / Course List

### 第一学期 / First Semester

#### 实变函数 / Real Analysis
- **课程简介**: 测度论基础、可测函数、勒贝格积分理论
- **学分**: 4
- **资料位置**: `./实变函数/`

#### 泛函分析 / Functional Analysis
- **课程简介**: 赋范空间、内积空间、线性算子理论
- **学分**: 4
- **资料位置**: `./泛函分析/`

#### 抽象代数 / Abstract Algebra
- **课程简介**: 群论、环论、域论的深入学习
- **学分**: 4
- **资料位置**: `./抽象代数/`

#### 微分几何 / Differential Geometry
- **课程简介**: 曲线论、曲面论、微分几何基本定理
- **学分**: 3
- **资料位置**: `./微分几何/`

### 第二学期 / Second Semester

#### 数值分析 / Numerical Analysis
- **课程简介**: 数值计算方法、误差分析、插值与逼近
- **学分**: 3
- **资料位置**: `./数值分析/`

#### 偏微分方程 / Partial Differential Equations
- **课程简介**: 偏微分方程的现代理论和方法
- **学分**: 3
- **资料位置**: `./偏微分方程/`

#### 拓扑学 / Topology
- **课程简介**: 点集拓扑、拓扑空间的基本概念和性质
- **学分**: 3
- **资料位置**: `./拓扑学/`

#### 专业选修课 / Elective Courses
- **课程内容**: 根据个人兴趣选择，如微分流形、李群与李代数、代数拓扑等
- **资料位置**: `./选修课/`

## 💡 学习建议 / Study Tips

### 实变函数 / Real Analysis
- 理解测度的定义和性质
- 掌握勒贝格积分与黎曼积分的区别
- 注重反例的构造和理解

### 泛函分析 / Functional Analysis
- 建立抽象空间的概念
- 理解三大基本定理（一致有界原理、开映射定理、闭图像定理）
- 注意有限维与无限维的本质区别

### 抽象代数 / Abstract Algebra
- 理解群、环、域的结构
- 掌握同态与同构的概念
- 注重代数结构之间的联系

### 微分几何 / Differential Geometry
- 培养几何直观
- 理解活动标架法
- 掌握高斯曲率和平均曲率

### 数值分析 / Numerical Analysis
- 理解数值方法的稳定性和收敛性
- 掌握误差分析方法
- 注重算法的实现

### 拓扑学 / Topology
- 理解开集公理系统
- 掌握连续映射和同胚的概念
- 建立拓扑不变量的思想

## 🎯 考试重点 / Exam Focus

### 实变函数
- 测度的定义和性质
- 可测函数的判定
- 勒贝格积分的计算
- 收敛定理（单调收敛、控制收敛、Fatou引理）

### 泛函分析
- 赋范空间的完备性
- 有界线性算子和有界线性泛函
- Hahn-Banach定理
- 三大基本定理

### 抽象代数
- 群的基本性质和子群
- 正规子群和商群
- 环和理想
- 域扩张和伽罗瓦理论基础

### 微分几何
- 曲线的曲率和挠率
- 曲面的第一、第二基本形式
- 高斯曲率和平均曲率
- 测地线

### 数值分析
- 插值方法（拉格朗日、牛顿、样条）
- 数值积分（梯形、Simpson）
- 线性方程组的迭代法
- 非线性方程求根

### 拓扑学
- 拓扑空间的基本概念
- 连续映射和同胚
- 紧致性、连通性、分离性
- 商空间和乘积空间
