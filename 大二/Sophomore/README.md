# 大二课程 / Sophomore Year Courses

## 📚 课程列表 / Course List

### 第一学期 / First Semester

#### 数学分析 III / Mathematical Analysis III
- **课程简介**: 多元函数微积分，包括偏导数、重积分、曲线积分等
- **学分**: 5
- **资料位置**: `./数学分析/`

#### 常微分方程 / Ordinary Differential Equations
- **课程简介**: 研究微分方程的理论与解法，包括一阶方程、高阶线性方程、方程组等
- **学分**: 4
- **资料位置**: `./常微分方程/`

#### 概率论与数理统计 / Probability Theory and Mathematical Statistics
- **课程简介**: 概率论基础理论、随机变量及其分布、数理统计基础
- **学分**: 4
- **资料位置**: `./概率论与数理统计/`

### 第二学期 / Second Semester

#### 复变函数 / Complex Analysis
- **课程简介**: 复数域上的分析学，包括解析函数、复积分、留数定理等
- **学分**: 4
- **资料位置**: `./复变函数/`

#### 数学物理方程 / Mathematical Physics Equations
- **课程简介**: 偏微分方程的基本理论和求解方法，包括分离变量法、特征线法等
- **学分**: 3
- **资料位置**: `./数学物理方程/`

#### 抽象代数基础 / Introduction to Abstract Algebra
- **课程简介**: 群、环、域的基础知识
- **学分**: 3
- **资料位置**: `./抽象代数/`

## 💡 学习建议 / Study Tips

### 常微分方程 / ODE
- 掌握各类方程的标准求解方法
- 理解解的存在唯一性定理
- 注意定性理论的基本思想

### 复变函数 / Complex Analysis
- 理解解析函数的本质
- 熟练掌握复积分的计算
- 重视留数定理及其应用

### 概率论与数理统计 / Probability and Statistics
- 理解概率的公理化定义
- 掌握常见分布及其性质
- 注意概率论与实际问题的联系

### 数学物理方程 / Mathematical Physics Equations
- 理解三类典型方程（波动、热传导、拉普拉斯）的物理背景
- 掌握分离变量法的应用
- 注意边界条件和初值条件的作用

## 🎯 考试重点 / Exam Focus

### 常微分方程
- 一阶方程的求解方法
- 高阶线性方程的通解结构
- 常系数线性方程组的求解
- 解的稳定性分析

### 复变函数
- 解析函数的判定
- 复积分计算（柯西定理）
- 级数展开（泰勒级数、洛朗级数）
- 留数计算及应用

### 概率论与数理统计
- 概率计算（古典概型、条件概率）
- 随机变量及其分布
- 数字特征（期望、方差）
- 大数定律和中心极限定理

### 数学物理方程
- 方程的分类和化简
- 分离变量法
- 行波法和特征线法
- 格林函数法
