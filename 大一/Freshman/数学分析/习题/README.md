# 习题 / Exercises

本文件夹用于存放课程习题及解答。

This folder is for course exercises and solutions.

## 📝 习题资料要求 / Exercise Requirements

### 内容要求 / Content Requirements
- 题目来源清晰（课本、习题集等）
- 解答过程完整
- 步骤清晰，易于理解
- 注明难度等级（如有）

### 格式建议 / Format Suggestions
- **PDF**: 手写解答扫描或排版好的解答
- **Markdown + LaTeX**: 便于编辑和版本控制
- **LaTeX源文件**: 对于数学公式较多的习题

### 命名规范 / Naming Convention
```
题源-章节-题号范围-作者-年份.格式
例如: 教材-第一章-习题1-20解答-2024.pdf
      Textbook-Ch1-Ex1-20-Solutions-2024.pdf
      吉米多维奇-1001-1050题解答.pdf
```

## 📚 习题类型 / Types of Exercises

- **课后习题** / Textbook Exercises: 教材课后习题
- **补充习题** / Supplementary Exercises: 老师布置的额外习题
- **历年考题** / Past Exam Problems: 往年考试题目
- **竞赛题** / Competition Problems: 数学竞赛题目

## 💡 做题建议 / Problem-solving Tips

1. **独立思考**: 先独立尝试解题，再参考答案
2. **多种方法**: 尝试用不同方法解同一道题
3. **总结归纳**: 总结题型和解题技巧
4. **错题整理**: 建立错题本，记录易错点
5. **难题标记**: 标记难题，定期复习

## ⚠️ 注意事项 / Important Notes

- 上传他人的解答时，请注明出处
- 不要上传未公开的试卷答案
- 确保解答的正确性
- 欢迎提供多种解法
