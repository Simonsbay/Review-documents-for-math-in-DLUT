# 数学分析 / Mathematical Analysis

## 📖 课程信息 / Course Information

- **课程名称**: 数学分析 I, II, III
- **学分**: 5 + 5 + 5 = 15
- **开课学期**: 大一上、大一下、大二上
- **先修课程**: 高中数学

## 📚 课程内容 / Course Content

### 数学分析 I (Mathematical Analysis I)
1. **实数理论** / Real Number Theory
2. **数列极限** / Limits of Sequences
3. **函数极限与连续性** / Function Limits and Continuity
4. **导数与微分** / Derivatives and Differentials
5. **微分中值定理及应用** / Mean Value Theorems and Applications
6. **不定积分** / Indefinite Integrals
7. **定积分** / Definite Integrals

### 数学分析 II (Mathematical Analysis II)
1. **定积分应用** / Applications of Definite Integrals
2. **数项级数** / Series of Numbers
3. **函数列与函数项级数** / Sequences and Series of Functions
4. **幂级数** / Power Series
5. **傅里叶级数** / Fourier Series

### 数学分析 III (Mathematical Analysis III)
1. **多元函数的极限与连续** / Limits and Continuity of Multivariable Functions
2. **偏导数与全微分** / Partial Derivatives and Total Differentials
3. **多元函数的极值** / Extrema of Multivariable Functions
4. **重积分** / Multiple Integrals
5. **曲线积分与曲面积分** / Line Integrals and Surface Integrals
6. **场论初步** / Introduction to Field Theory

## 💡 学习建议 / Study Tips

### 基础阶段 (Mathematical Analysis I)
1. **掌握 ε-δ 语言**：这是数学分析的核心，要反复练习直到熟练
2. **重视极限计算**：各种极限的计算技巧要熟练掌握
3. **理解连续性**：连续、一致连续、间断点的概念要清楚
4. **导数与微分**：几何意义和物理意义都要理解

### 提高阶段 (Mathematical Analysis II)
1. **级数理论**：各种收敛性判别法要熟练运用
2. **函数项级数**：一致收敛的概念很重要
3. **幂级数展开**：常见函数的幂级数展开要记住

### 进阶阶段 (Mathematical Analysis III)
1. **偏导数与全微分**：理解多元函数的微分学
2. **重积分计算**：掌握各种坐标变换和累次积分
3. **格林公式、高斯公式、斯托克斯公式**：理解并能熟练应用

## 🎯 考试重点 / Exam Focus

### 期中考试 (Midterm)
- 极限的 ε-N、ε-δ 证明
- 连续性与一致连续性的判定
- 导数与微分的计算
- 微分中值定理的应用

### 期末考试 (Final)
- 积分计算（不定积分、定积分）
- 级数的收敛性判别
- 函数项级数的一致收敛性
- 多元函数的极值问题
- 重积分的计算

## ⚠️ 常见难点 / Common Difficulties

1. **极限的严格证明**：需要掌握 ε-δ 语言
2. **一致连续与连续的区别**：要通过大量例子理解
3. **级数的条件收敛**：收敛性判别是难点
4. **重积分的坐标变换**：雅可比行列式的应用
5. **曲线积分与路径无关的条件**

## 💪 练习建议 / Practice Suggestions

1. **每天练习极限计算**：熟能生巧
2. **定理证明要动手写**：不能只看不写
3. **错题本很重要**：记录易错点
4. **参加习题课**：认真听讲，多提问
