# 试卷 / Exams

本文件夹用于存放往年试卷及相关资料。

This folder is for past exam papers and related materials.

## 📝 试卷资料要求 / Exam Requirements

### 可上传内容 / Acceptable Content
- ✅ 已公开的往年试卷
- ✅ 期中、期末考试题目
- ✅ 模拟试卷
- ✅ 试卷解答（注明来源）

### 禁止上传内容 / Prohibited Content
- ❌ 未公开的试卷
- ❌ 未经授权的试卷答案
- ❌ 泄露的考试题目

### 格式建议 / Format Suggestions
- **PDF**: 扫描件或排版文件
- **注明年份和学期**: 如 2024年春季学期期末试卷

### 命名规范 / Naming Convention
```
年份-学期-考试类型.格式
例如: 2024春-期末试卷.pdf
      2023秋-期中试卷-含答案.pdf
      2024-Spring-Final-Exam.pdf
```

## 💡 复习建议 / Review Tips

### 使用往年试卷的方法 / How to Use Past Exams
1. **了解题型**: 熟悉考试的题型和难度
2. **模拟考试**: 按考试时间要求完成试卷
3. **总结考点**: 分析考试重点和高频考点
4. **查漏补缺**: 找出薄弱环节，针对性复习

### 考前准备 / Exam Preparation
1. **系统复习**: 不要只做真题，要系统复习知识点
2. **计时练习**: 提高答题速度
3. **注意细节**: 注意答题格式和步骤的完整性
4. **调整心态**: 保持良好的心态
