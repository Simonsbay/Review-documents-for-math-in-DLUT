# 笔记 / Notes

本文件夹用于存放课程笔记。

This folder is for course notes.

## 📝 笔记要求 / Notes Requirements

### 内容要求 / Content Requirements
- 内容完整，覆盖重要知识点
- 逻辑清晰，便于理解
- 公式准确，排版整洁

### 格式建议 / Format Suggestions
- **PDF**: 手写笔记扫描或排版好的笔记
- **Markdown**: 文本型笔记，便于编辑和版本控制
- **LaTeX源文件**: 对于数学公式较多的笔记

### 命名规范 / Naming Convention
```
课程名-章节-笔记类型-作者-年份.格式
例如: 数学分析-第一章-课堂笔记-张三-2024.pdf
      Mathematical_Analysis-Ch1-Lecture_Notes-2024.pdf
```

## 📚 笔记类型 / Types of Notes

- **课堂笔记** / Lecture Notes: 上课时的记录
- **整理笔记** / Organized Notes: 课后整理的系统笔记
- **复习笔记** / Review Notes: 考前复习用的精简笔记
- **思维导图** / Mind Maps: 知识结构图

## 💡 笔记技巧 / Note-taking Tips

1. **及时整理**: 课后尽快整理笔记，趁记忆清晰
2. **标注重点**: 用不同颜色或符号标注重点内容
3. **补充说明**: 添加自己的理解和例子
4. **交叉引用**: 关联相关知识点
5. **定期回顾**: 定期复习笔记，加深印象
