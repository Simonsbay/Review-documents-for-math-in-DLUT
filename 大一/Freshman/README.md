# 大一课程 / Freshman Year Courses

## 📚 课程列表 / Course List

### 第一学期 / First Semester

#### 数学分析 I / Mathematical Analysis I
- **课程简介**: 数学专业最重要的基础课程之一，主要内容包括极限理论、连续性、微分学等
- **学分**: 5
- **资料位置**: `./数学分析/`

#### 高等代数 I / Advanced Algebra I
- **课程简介**: 数学专业另一门重要基础课，主要内容包括多项式理论、线性方程组、矩阵理论等
- **学分**: 5
- **资料位置**: `./高等代数/`

#### 解析几何 / Analytic Geometry
- **课程简介**: 用代数方法研究几何问题，包括向量、平面与直线、二次曲面等
- **学分**: 3
- **资料位置**: `./解析几何/`

### 第二学期 / Second Semester

#### 数学分析 II / Mathematical Analysis II
- **课程简介**: 继续学习数学分析，主要内容包括积分学、级数理论等
- **学分**: 5
- **资料位置**: `./数学分析/`

#### 高等代数 II / Advanced Algebra II
- **课程简介**: 继续学习高等代数，主要内容包括线性空间、线性变换、欧氏空间等
- **学分**: 5
- **资料位置**: `./高等代数/`

## 💡 学习建议 / Study Tips

### 数学分析 / Mathematical Analysis
- 重视基础概念的理解，特别是极限的 ε-δ 定义
- 多做习题，提高计算能力
- 注意定理的证明思路和技巧

### 高等代数 / Advanced Algebra
- 理解线性空间和线性变换的抽象概念
- 掌握矩阵运算的技巧
- 注重理论与计算的结合

### 解析几何 / Analytic Geometry
- 培养空间想象能力
- 掌握向量运算的熟练程度
- 理解代数与几何的对应关系

- 平面与直线方程
