# 快速开始指南 / Quick Start Guide

欢迎使用大连理工大学数学专业课程复习资料仓库！

Welcome to the DLUT Math Major Course Review Materials Repository!

## 🎯 使用指南 / User Guide

### 作为学习者 / As a Learner

如果你是来寻找学习资料的：

If you're here to find study materials:

1. **浏览课程** / Browse Courses
   - 根据你的年级，进入对应的文件夹（大一、大二、大三、大四）
   - 找到你需要的课程文件夹
   - 查看该课程的 README.md 了解课程信息

2. **下载资料** / Download Materials
   - 直接点击文件下载
   - 或者 Clone 整个仓库到本地：
   ```bash
   git clone https://github.com/Simonsbay/Review-documents-for-math-in-DLUT.git
   ```

3. **参与讨论** / Join Discussions
   - 在 Issues 中提问题
   - 在 Discussions 中交流学习心得

### 作为贡献者 / As a Contributor

如果你想分享自己的资料：

If you want to share your materials:

1. **阅读贡献指南** / Read Contributing Guide
   - 仔细阅读 [CONTRIBUTING.md](CONTRIBUTING.md)
   - 了解资料格式和命名规范

2. **准备资料** / Prepare Materials
   - 整理你的笔记、习题解答等
   - 转换为推荐格式（PDF、Markdown）
   - 确保文件大小合适（大文件使用 Git LFS）

3. **提交 Pull Request** / Submit Pull Request
   - Fork 本仓库
   - 添加你的资料到对应位置
   - 提交 PR 并描述你的贡献

4. **等待审核** / Wait for Review
   - 维护者会审核你的 PR
   - 根据反馈进行修改
   - 合并后你的贡献将帮助更多同学！

## 🌟 推荐学习路径 / Recommended Learning Path

### 大一新生 / Freshmen
1. 从 `大一/Freshman/` 开始
2. 重点关注**数学分析**和**高等代数**
3. 建立扎实的数学基础

### 大二学生 / Sophomores
1. 复习大一内容，查漏补缺
2. 学习 `大二/Sophomore/` 的课程
3. 开始接触更深入的数学理论

### 大三学生 / Juniors
1. 进入专业课学习阶段
2. 参考 `大三/Junior/` 的资料
3. 培养抽象思维能力

### 大四学生 / Seniors
1. 准备毕业论文
2. 查看 `大四/Senior/` 的资源
3. 为升学或就业做准备

## 💡 学习建议 / Study Tips

1. **系统学习** / Systematic Study
   - 不要跳跃式学习，按部就班
   - 打好基础再学习高级内容

2. **多做练习** / Practice More
   - 数学需要大量练习
   - 利用仓库中的习题资源

3. **理解为主** / Focus on Understanding
   - 不要死记硬背
   - 理解概念的本质

4. **交流分享** / Communicate and Share
   - 与同学讨论问题
   - 分享自己的理解

## 🔧 常见问题 / FAQ

### Q1: 如何找到特定课程的资料？
**A**: 根据课程所在年级，进入对应文件夹，然后找到课程名称文件夹。

### Q2: 我可以直接下载单个文件吗？
**A**: 可以。在 GitHub 上点击文件，然后点击 "Download" 按钮。

### Q3: 资料可以商用吗？
**A**: 不可以。本仓库资料仅供学习参考，禁止商业使用。

### Q4: 我想贡献资料，但不会用 Git 怎么办？
**A**: 可以通过 Issues 联系维护者，或者学习 Git 基础操作（推荐）。

### Q5: 资料有错误怎么办？
**A**: 请在 Issues 中报告错误，或者提交 PR 修正错误。
